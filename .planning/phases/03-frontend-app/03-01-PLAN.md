1. Create `src/hooks/useSearchIndex.ts` — "use client" hook:
   - useState for `index: IndexEntry[] | null` and `loading: boolean` (initially true)
   - useEffect: check localStorage key `santescope_index` — if present and ts < 24h old, use cached data
   - Otherwise fetch `/data/index.json`, parse as IndexEntry[], store in localStorage as `{data, ts: Date.now()}`
   - Return `{ index, loading }`
   - Per D-09: skeleton while loading, localStorage cache

2. Create `src/lib/search.ts` — filterIndex function:
   - Signature: `export function filterIndex(index: IndexEntry[], query: string): IndexEntry[]`
   - Normalize both query and entry.nom: `.toLowerCase().normalize("NFD").replace(/\p{Diacritic}/gu, "")`
   - Two-pass: startsWith matches first, then includes matches (per D-08)
   - Also match on entry.code.startsWith(query) for code-based search
   - Early exit: stop when starts.length + includes.length >= 8
   - Return max 8 results: `[...starts, ...includes].slice(0, 8)`

3. Create `src/components/search/SearchBar.tsx` — "use client" component:
   - Props: `{ fullScreen?: boolean }` (fullScreen for landing, compact for header)
   - Uses useSearchIndex() hook
   - Input with 150ms debounce (useRef + setTimeout pattern, per D-08)
   - While loading: render shadcn Skeleton component in place of input (per D-29)
   - Dropdown: position absolute below input, show filtered results
   - Each result row: ScoreBadge (small 28px square with DPE color + letter) + "{nom} ({dept})" + population
   - On result click: `router.push(/commune/${entry.code})` using next/navigation useRouter
   - Empty state when query > 0 chars but 0 results: "Aucune commune trouvée — vérifiez le nom ou le code INSEE" (per UI-SPEC copywriting)
   - Keyboard: arrow keys navigate results, Enter selects highlighted
   - Use shadcn Command component (cmdk) as the base for accessible keyboard navigation
   - Score badge inline: small colored square (20x16px) with letter, background from DPE_COLORS[classe], text from DPE_TEXT_COLORS[classe]. If classe is null, show gray badge "–".
