## Decisions Made

1. **Removed nested .git** — create-next-app initializes its own git repo; removed to incorporate into parent monorepo
2. **public/data excluded from git** — 34,969 commune JSONs + 2.9MB index.json are generated by notebooks; only source code tracked
3. **params is a Promise in Next.js 15** — dynamic route `params` must be `await`ed in server components (breaking change from Next 14)
4. **DPE badge uses inline style** — Tailwind cannot generate dynamic classes for runtime color values from the DPE_COLORS map
5. **localStorage cache 24h TTL** — avoids re-fetching 2.9MB index.json on each page load
