2. **PdfDownloadButton.tsx** — "use client" (per D-22):
   - Props: `{ communeCode: string; communeNom: string }`
   - Renders shadcn Button variant="outline": "Télécharger le rapport" with lucide-react Download icon
   - On click:
     ```typescript
     const el = document.getElementById("pdf-export-root");
     if (!el) return;
     const html2canvas = (await import("html2canvas")).default;
     const canvas = await html2canvas(el, {
       scale: 2,
       useCORS: true,
       backgroundColor: "#1a1a1a",
       logging: false,
     });
     const link = document.createElement("a");
     link.download = `santescope-${communeCode}-${new Date().toISOString().slice(0,10)}.png`;
     link.href = canvas.toDataURL("image/png");
     link.click();
     ```
   - MUST dynamically import html2canvas (not top-level import) to avoid SSR "window is not defined" error (pitfall from RESEARCH.md)
   - Show loading state on button while capturing (disable button, show spinner)
