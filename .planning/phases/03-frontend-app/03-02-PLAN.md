1. **ScoreBadge.tsx** (per D-11):
   - Props: `{ classe: string | null; size?: number }` (default size=48)
   - Render a square div with `borderRadius: 8px`, background from `DPE_COLORS[classe]`, centered text from `DPE_TEXT_COLORS[classe]`, font-size = size * 0.45, font-weight 700
   - If `classe === null`: gray background (#CBD5E1), text "–"
   - Port directly from mockup `ScoreBadge` function, converting inline styles to Tailwind where possible

2. **DpeStrip.tsx** (per D-11):
   - Props: `{ active: string | null; small?: boolean }`
   - Render 5 small tiles A-E, each 24x18px (small: 20x15px), with DPE color backgrounds
   - Active tile: `border: 2px solid #0F766E`, opacity 1. Others: `border: 2px solid transparent`, opacity 0.5
   - If active is null, no tile is highlighted

3. **ScoreGauge.tsx** (per D-11):
   - Props: `{ classe: string | null; score: number | null }`
   - Horizontal bar: full-width gray track (#f1f5f9), filled portion width from BAR_W[classe], color from DPE_COLORS[classe]
   - Beside bar: score text "{score}/10" in 20px semibold. If score null: "Données insuffisantes" in slate-400
   - Below: DPE_LABELS[classe] classification label (per UI-SPEC copywriting: "Très vulnérable", "Vulnérable", "Situation modérée", "Situation correcte", "Situation exemplaire")

4. **ScoreDetail.tsx** (per D-12):
   - Props: `{ scoreDetail: CommuneData["score_detail"] }`
   - 4 rows, one per component:
     - "Accès aux soins (APL)" — value: `scoreDetail.apl.toFixed(1)`, national: `scoreDetail.apl_national.toFixed(1)`, unit: "pour 100K hab."
     - "Taux de pauvreté" — value: `scoreDetail.pauvrete !== null ? (scoreDetail.pauvrete * 100).toFixed(1) + "%" : "Non disponible"`, national: `(scoreDetail.pauvrete_national * 100).toFixed(1) + "%"`
     - "Personnes 75+ isolées" — value: `(scoreDetail.pct_75_seuls * 100).toFixed(1) + "%"`, national: `(scoreDetail.pct_75_seuls_national * 100).toFixed(1) + "%"`
     - "Temps d'accès urgences" — value: `scoreDetail.temps_urgences_min.toFixed(0) + " min"`, national: `scoreDetail.temps_urgences_national.toFixed(0) + " min"`
   - Each row: label (left), value (center), comparison arrow + delta vs national (right). Red text if worse than national, green if better. Use lucide-react ArrowUp/ArrowDown icons.
   - If pauvrete is null, gray out that row with "Non disponible" text

5. **DominoAlert.tsx** (per D-14):
   - Props: `{ domino: NonNullable<CommuneData["domino"]> }`
   - Container: `rounded-lg border-l-4 p-4`
   - If `domino.pct_55plus > 0.5`: `bg-red-50 border-red-400` (critical)
   - Else: `bg-orange-50 border-orange-400` (warning)
   - Title: "Alerte succession médicale" — semibold, orange-900 or red-900
   - Line 1: `"{Math.round(domino.pct_55plus * 100)}% des médecins ont 55+ ans (moy. dept : {Math.round(domino.dept_avg * 100)}%)"`
   - Line 2: `"Estimation 2030 : -{domino.pertes_2030} médecins"`

6. **MissingDoctors.tsx** (per D-15):
   - Props: `{ manques: string[]; pathologies: CommuneData["pathologies_dept"] }`
   - Map specialty names to pathology keys: `{Endocrinologue: "diabete", Cardiologue: "cardiovasculaire", Psychiatre: "psychiatrique", Oncologue: "cancers", Pneumologue: "respiratoire"}`
   - For each manque string, if it matches a pathology key, show: "• {manque} — {pathologyLabel}: {value.toFixed(1)}" (raw prevalence rate from pathologies_dept)
   - If no pathology match, show: "• {manque}"
   - Max 5 items, then "+{remaining} autres" link
   - Section title: "Spécialités manquantes" in semibold

7. **MiniMap.tsx** (per D-19, UI-08):

   - SVG France silhouette (simplified path from mockup) at opacity 0.15
   - Pin dot: 10px teal circle with white border at center
   - Label below pin: commune name in 10px semibold teal
